# Album Browser

Angular SPA for Lab 6 — Routing, HTTP & Services.

## Setup & Run

```bash
npm install
ng serve
```

Then open [http://localhost:4200](http://localhost:4200).

## Features

- Browse 100 albums fetched from JSONPlaceholder API
- View album details and edit the title
- Browse photos for any album in a responsive grid
- Delete albums from the list
- Navigation with active link highlighting
